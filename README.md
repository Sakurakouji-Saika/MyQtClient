# QT_MyClient

这是一个基于Qt框架开发的客户端应用程序，主要用于实现用户登录、好友管理、消息发送与接收、文件传输以及个人资料设置等即时通讯功能。

## 功能特性

- **用户登录与注册**：用户可以登录到服务器或注册新账户。
- **好友管理**：支持添加、删除好友，查看好友列表及状态。
- **消息收发**：支持即时消息的发送与接收，包含聊天记录存储。
- **文件传输**：提供文件上传与下载功能，支持头像更换。
- **个人资料设置**：允许用户设置头像、昵称等个人信息。
- **界面美观**：采用现代UI设计，支持样式定制。

## 技术栈

- **C++**：主要编程语言。
- **Qt**：用于构建图形用户界面和网络通信。
- **SQLite**：本地数据库用于存储用户数据和聊天记录。
- **CMake**：构建系统。

## 目录结构

- `src/`：源代码目录，包含所有C++源文件。
- `src/DataBaseManage/`：数据库管理模块，处理与SQLite数据库交互。
- `src/Network/`：网络通信模块，处理客户端与服务器之间的消息和文件传输。
- `src/app/`：主应用程序窗口和登录窗口的实现。
- `src/utils/`：工具类和通用函数。
- `src/widgets/`：自定义控件。
- `ui/components/`：UI组件，如聊天窗口、好友列表等。
- `resources/`：资源文件，包括样式表、图片和SVG图标。

## 构建与运行

1. 克隆仓库到本地：
   ```bash
   git clone https://gitee.com/Sirin/QT_MyClient.git
   ```

2. 使用Qt Creator打开项目并构建：
   - 打开Qt Creator，选择“文件”->“打开文件或项目”。
   - 导航到克隆的仓库目录并选择`CMakeLists.txt`。
   - 构建并运行项目。

## 使用说明

1. 启动应用程序后，输入服务器地址和端口进行连接。
2. 登录已有账户或注册新账户。
3. 使用好友管理功能添加或删除好友。
4. 与好友进行即时通讯，发送和接收文件。
5. 在设置中更改个人资料和头像。
